Detailed Description

LogMessage Parameters:

Level is what level the log message is message is the log message itself category is for what system like renderer, physics, etc. file is the source file where the log was generated line is the line number in the source file function is the function name where the log was generated threadId is the ID of the thread that generated the log timestamp is the time when the log was generated
