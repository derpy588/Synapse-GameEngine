\doxysubsection{Detailed Description}
\doxylink{struct_synapse_1_1_log_message}{Log\+Message} Parameters\+:

Level is what level the log message is message is the log message itself category is for what system like renderer, physics, etc. file is the source file where the log was generated line is the line number in the source file function is the function name where the log was generated thread\+Id is the ID of the thread that generated the log timestamp is the time when the log was generated
